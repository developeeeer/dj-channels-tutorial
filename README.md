# Django-Channels で Chat アプリを作成

## https://channels.readthedocs.io/en/latest/tutorial/part_1.html

1. pipenv で仮想環境構築

プロジェクトディレクトリを作成

```terminal
mkdir project
```

pipenv 環境構築

```terminal
1. バージョン指定無し
pipenv install

2. バージョン指定有り
pipenv --python 3.9.9
```

package インストール

```terminal
pipenv shell
pip install Django==3.2.12

# 仮想環境を非アクティブ化にする
exit
```
